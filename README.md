\# ЁЯОо Arcade Hub



A C-based arcade gaming platform featuring multiple classic games integrated into a single application.



\---



\## ЁЯУМ Overview



Arcade Hub is a multi-game project developed using C Programming. The project combines multiple classic arcade games into one platform while demonstrating core concepts of Data Structures, Algorithms, File Handling, and Modular Programming.



The objective of this project is to provide an engaging gaming experience while applying fundamental computer science concepts in a practical environment.

\* ЁЯзй 2048
