# 🎮 Arcade Hub



A C-based arcade gaming platform featuring multiple classic games integrated into a single application.



---



## 📌 Overview



Arcade Hub is a multi-game project developed using C Programming. The project combines multiple classic arcade games into one platform while demonstrating core concepts of Data Structures, Algorithms, File Handling, and Modular Programming.



The objective of this project is to provide an engaging gaming experience while applying fundamental computer science concepts in a practical environment.



### Included Games



* 🧩 2048

* 🧠 Memory Game

* 🏃 Subway Runner



---



## ✨ Features



* Multiple games integrated into one project

* Modular and maintainable code structure

* Console-based user interface

* Score tracking and leaderboard systems

* File handling for saving game data

* Efficient game logic implementation

* Easy to expand with additional games

## 🚀 How to Run



1\. Open the project in your preferred C development environment.

2\. Compile the source files using GCC or any compatible C compiler.

3\. Run the generated executable.

4\. Select a game from the menu and start playing.



Example:



```bash

gcc main.c -o ArcadeHub

ArcadeHub.exe

```
